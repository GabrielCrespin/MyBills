# My Bills 💰

My Bills is a personal finance management web application designed to help users organize and track their monthly bills and expenses.
The system allows users to register bills and manage their expenses in a simple and intuitive way.

This project was created as a portfolio application to practice full-stack development using modern web technologies.

---

## 🚀 Features

* Register new bills and expenses
* View all registered bills
* Update existing bills
* Delete bills
* Monitor and organize monthly expenses
* Simple and intuitive interface

## ⚙️ How to Run the Project

### 1️⃣ Clone the repository

```bash
git clone https://github.com/GabrielCrespin/MyBills.git
```

### 2️⃣ Run the Backend

Navigate to the backend folder and run:

```bash
dotnet run
```

The API will start locally and can be tested using Swagger.

### 3️⃣ Run the Frontend

Navigate to the frontend folder and install dependencies:

```bash
npm install
```

Start the application:

```bash
npm start
```

## 📊 API Endpoints (Example)

| Method | Endpoint    | Description       |
| ------ | ----------- | ----------------- |
| GET    | /bills      | List all bills    |
| GET    | /bills/{id} | Get bill by ID    |
| POST   | /bills      | Create a new bill |
| PUT    | /bills/{id} | Update a bill     |
| DELETE | /bills/{id} | Delete a bill     |
